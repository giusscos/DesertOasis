Editable sources: `Developer/blender/DesertOasis/blend_sources/*.blend` (each contains the rig plus all

blender --background --factory-startup --python asset_scripts/gen_lobby.py

## Characters

All characters share the standard armature: `root, hips, spine, chest, neck,
head, shoulder/upperarm/forearm/hand _L/_R, thigh/shin/foot _L/_R`.

**Important deviation from the original prompt:** USDZ has no concept of
multiple named animation clips per file — a USD stage carries a single
animation timeline. `animationPlayer(forKey: "walk")` on a multi-clip USDZ is
not achievable directly, so each action ships as its own file (the pattern the
May 2026 assets already used):

| Base file (plays idle) | Per-action files |
|---|---|
| `player_man.usdz` | `player_man_{idle,walk,talk,wave}.usdz` |
| `player_woman.usdz` | `player_woman_{idle,walk,talk,wave}.usdz` |
| `npc_wanderer.usdz` | `npc_wanderer_{idle,talk,gesture}.usdz` |
| `npc_merchant.usdz` | `npc_merchant_{idle,talk,gesture}.usdz` |
| `npc_child.usdz` | `npc_child_{idle,talk,gesture}.usdz` |
| `npc_elder.usdz` | `npc_elder_{idle,talk,gesture}.usdz` |
| `npc_lost.usdz` | `npc_lost_{idle,talk,gesture}.usdz` |

Action lengths: idle 60 f (players) / 30 f (NPCs), walk 30 f loop, talk 90 f /
60 f, wave 45 f, gesture 45 f — all authored at 24 fps and loop-safe.

### SceneKit loading pattern

```swift
// Load the model once, then graft animations from the per-action files.
func loadCharacter(_ name: String, actions: [String]) -> SCNNode {
    let scene = SCNScene(named: "\(name).usdz")!
    let node = scene.rootNode.clone()
    for action in actions {
        let animScene = SCNScene(named: "\(name)_\(action).usdz")!
        animScene.rootNode.enumerateHierarchy { child, _ in
            for key in child.animationKeys {
                if let player = child.animationPlayer(forKey: key) {
                    player.stop()
                    node.addAnimationPlayer(player, forKey: action)
                }
            }
        }
    }
    return node
}
// Usage:
let man = loadCharacter("player_man", actions: ["idle", "walk", "talk", "wave"])
man.animationPlayer(forKey: "walk")?.play()
```

## Lobby props

- `lobby_tent.usdz` — 8 × 10 × 4 m A-frame shell, entrance on +Z (SceneKit),
  hanging emissive lantern, `sign_text` child node with carved "Desert Oasis".
- `lobby_bed.usdz` — cot with `diary_0` (red/gold, label), `diary_1`
  (green/silver), `diary_2` (blue/bronze). The 30-frame `open` animation
  (cover swings, pages fan) is baked on every diary's hinge nodes; grab the
  animation players from the tapped diary's subtree only.
- `lobby_table.usdz` — folding table with child nodes `node_compass`,
  `node_map`, `node_lantern`, `node_quill`, `node_hourglass` (60 f looping
  sand-flow animation).

## Environment props

`prop_cactus_saguaro` (2.5 m), `prop_cactus_barrel` (0.6 m),
`prop_rock_small`, `prop_rock_cluster`, `prop_palm_tree` (5 m, rigged trunk,
120 f `sway`), `prop_dead_tree`, `prop_oasis_water` (10 m disc, 60 f blend-shape
ripple), `prop_tumbleweed` (30 f `roll`: one revolution + 1.8 m of travel along
+Z in SceneKit — translate the node or strip the translation channel for
in-place rolling), `prop_sand_dune` (8 × 2.5 m).

Materials are UsdPreviewSurface (converted from Principled BSDF); emissive
parts (lanterns) carry emission for bloom. Coordinate conversion: Blender Z-up
exports to USD Y-up automatically; characters face +Z in SceneKit terms.
